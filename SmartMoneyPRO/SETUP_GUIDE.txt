PART 1 — RUN LOCALLY (Your Computer)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Install Python 3
  → Go to python.org/downloads
  → Download Python 3.11 or newer
  → IMPORTANT: Check "Add Python to PATH" during install

STEP 2: Start the server
  → Open folder: SmartMoneyPRO
  → Right-click inside → "Open in Terminal" (or cmd)
  → Type this and press Enter:

      python server/app.py

  → You will see:
      ✓ Got prices for 15 instruments live
      ✓ Listening on http://0.0.0.0:8080

STEP 3: Open the app
  → Open Chrome or Edge
  → Go to: http://localhost:8080
  → App opens with REAL live prices ✓

PART 2 — ACCESS FROM ANYWHERE IN THE WORLD
         (Free hosting on Railway.app — 5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

This gives you a URL like: https://smartmoney-pro.up.railway.app
You can open it on your phone, tablet, anywhere — different WiFi, mobile data, etc.

── METHOD A: Railway.app (Recommended — Free) ──

STEP 1: Create free accounts
  → github.com → Sign Up (free)
  → railway.app → Sign Up with GitHub (free)

STEP 2: Upload to GitHub
  a) Go to github.com → Click "+" → "New repository"
  b) Name it: smartmoney-pro
  c) Click "Create repository"
  d) You'll see a page with upload instructions

  Upload your files:
  → Click "uploading an existing file"
  → Drag ALL files from SmartMoneyPRO folder
  → (Make sure to include server/app.py, index.html, css/, js/ folders)
  → Click "Commit changes"

STEP 3: Deploy on Railway
  a) Go to railway.app
  b) Click "New Project"
  c) Click "Deploy from GitHub repo"
  d) Select your "smartmoney-pro" repo
  e) Railway auto-detects Python and deploys!
  f) Wait 2-3 minutes
  g) Click "Settings" → "Domains" → "Generate Domain"
  h) You get a URL like: https://smartmoney-pro-production.up.railway.app

STEP 4: Open from anywhere
  → Share that URL with your phone
  → Open on any device, any network, anywhere in the world
  → All data (signals, account, win rate) stored on Railway server
  → Prices update from real APIs every 30 seconds

── METHOD B: Render.com (Also Free) ──

  a) Go to render.com → Sign Up
  b) New → Web Service → Connect GitHub
  c) Select your repo
  d) Build Command: (leave empty)
  e) Start Command: python server/app.py
  f) Click "Create Web Service"
  g) Get your URL in 5 minutes

PART 3 — ADD YOUR ANTHROPIC API KEY (For AI Analyzer)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The AI Chart Analyzer uses Claude. You need a free API key.

STEP 1: Get API key
  → Go to console.anthropic.com
  → Sign up (free trial credits included)
  → API Keys → Create Key
  → Copy the key (starts with sk-ant-)

STEP 2: Add to Railway (for global use)
  → Railway dashboard → Your project → Variables
  → Add variable:
      Name:  ANTHROPIC_API_KEY
      Value: sk-ant-your-key-here
  → Click "Add"
  → App redeploys automatically

STEP 3: For local use only
  Windows (cmd):
    set ANTHROPIC_API_KEY=sk-ant-your-key-here
    python server/app.py

  Mac/Linux (terminal):
    ANTHROPIC_API_KEY=sk-ant-your-key-here python server/app.py

  ✓ BTC/USD, ETH/USD, SOL/USD, BNB/USD, XRP/USD prices  → CoinGecko API

CALCULATED (not from external feed):

  ~ Fibonacci levels (math from real price)
  ~ Position size (math: 1% risk / SL distance)

The signals are based on REAL prices and REAL SMC math.
They are not random. They follow the exact rules from your PPT course.
